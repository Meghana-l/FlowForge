# FlowForge

Enterprise workflow intelligence platform that assesses current-state processes, applies zero-based AI-native redesign, quantifies transformation value, and generates multi-quarter roadmaps.

## What it does

Select any enterprise domain — Finance, HR, Procurement, IT, Sales, Legal, Customer Service, or Regulatory Reporting — then drill into a specific process. FlowForge maps the current workflow step by step, identifies inefficiency vectors, redesigns it from first principles using agentic AI, models the ROI, and outputs a phased transformation roadmap with governance controls.

## Features

- **Current-state assessment** — process flow visualization, cycle time, error rate, automation coverage, manual touchpoint analysis
- **AI-native redesign** — agentic orchestration blueprint with specific agent definitions and redesigned step-by-step flow
- **Value creation model** — annual cost savings, productivity gain, cycle time reduction, and a cumulative ROI chart across quarters
- **Transformation roadmap** — phased multi-quarter initiative plan with tagging by initiative type
- **AI governance** — risk rating and specific control framework for safe deployment

## Stack

- Vanilla HTML/CSS/JS frontend (no build step)
- Chart.js for ROI visualization
- Claude AI via serverless proxy for analysis generation
- Deployed on Vercel
